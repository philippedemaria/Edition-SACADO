
On tire au hasard une carte dans un jeu de 32 cartes. On note :
\begin{description}
\item[A :] l'événement : "La carte tirée est le roi de cœur".
\item[B :] l'événement : "La carte tirée est un roi".
\item[C :] l'événement : "La carte tirée est noire".
\end{description}
Calculer 
\begin{enumerate}
\item $p(A \cap B)$
\item $p(A \cup B)$
\item $p(A \cap C)$
\item $p(A \cup C)$
\end{enumerate}
